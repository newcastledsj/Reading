# 《如何阅读一本书》

## 第一篇  阅读的层次

### 第一章  阅读的活力与艺术

* 面向对象
   * 读书的主要目的是增进理解能力的人
* 阅读艺术
   * 这是一个凭借着头脑运作，除了玩味读物中的一些字句之外，不假任何外助，以一己之力来提升自我的过程。
* 如何读好一本书
    * 主动阅读 
    * 在阅读中使用技巧
* 主动阅读的要求
   * 观察
    * 记忆
    * 思考
    * 想象力
* 阅读的目标
    * 获得资讯：
        * 指导型的学习：行动立足于被传达的信息
    * 求得理解：
        * 自我发现型的学习：行动立足于自然或世界
* 为增进理解而阅读的要求
    * 一开始不相等的理解程度
    * 阅读的人一定要把不相等的理解力克服到一定程度之内
